# Study-Time-Tracker

## Project Description

StudyTimeTracker is a browser-based web application that helps students track their study time, manage their sessions, and monitor their learning progress.
The application allows users to record study sessions, organize them by subject, and view their study history.

---

## Project Goals / Features

* Track study sessions (start/stop or manual input)
* Store study sessions in a database
* View study history
* Calculate total study time
* Organize sessions by subject

---

##  User Stories

* As a student, I want to track my study time so that I can improve my productivity.
* As a student, I want to save my study sessions so that I do not lose my data.
* As a student, I want to view my study history so that I can analyze my study habits.
* As a student, I want to organize my sessions by subject so that I know what I study most.

---

## Technologies / Libraries

* Python
* NiceGUI (frontend)
* SQLite (database)


---

## Use Cases

1. The user enters a subject
2. The user starts or records a study session
3. The user stops the session
4. The session is saved in the database
5. The user can view all previous sessions
6. The system calculates total study time
